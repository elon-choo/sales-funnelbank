## 프로젝트 핵심 개요

### 만들려는 것
**마그네틱 세일즈 마스터클래스 수강생 전용 랜딩페이지 자동 제작 웹앱**

### 핵심 기능
1. **사용자 인증**: 이메일 가입 → 관리자 수동 승인 → 로그인
2. **AI 기획 도우미**: Claude 기반, 마그네틱 세일즈 철학 반영, 랜딩페이지 기획 대화
3. **랜딩페이지 빌더**: 프롬프트 + 이미지 → HTML 자동 생성
4. **DB 수집 상세페이지**: 구글 폼 연결, 고객 정보 수집용

### 타겟 사용자
보험 설계사, 피부샵 운영자, 코치/컨설턴트 등 1인 세일즈 종사자

### 기술 스택 (예정)
- Frontend: React + TypeScript + TailwindCSS
- Backend: Supabase (인증, DB, 스토리지)
- AI: Claude API
- 배포: Vercel
