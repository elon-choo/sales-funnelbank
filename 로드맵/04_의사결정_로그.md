### DEC-003: 인증 방식
| 항목 | 내용 |
|------|------|
| 일시 | 2025-12-15 |
| 결정 | 수동 승인 방식 |
| 근거 | 수강생만 사용하도록 제한 필요 |

**플로우:**
1. 사용자가 이메일/비밀번호로 가입 신청
2. `users` 테이블에 `approved: false`로 저장
3. 관리자가 Supabase 대시보드에서 `approved: true`로 변경
4. 승인된 사용자만 로그인 가능
