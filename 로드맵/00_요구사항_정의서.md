# 마그네틱 세일즈 랜딩페이지 자동 제작 웹앱 - 요구사항 정의서

## 1. 프로젝트 개요

### 1.1 프로젝트 배경
줄리아의 마그네틱 세일즈 마스터클래스 1기 수강생들 중 보험/피부샵 등 다양한 세일즈 종사자들이 **고객 DB 부족**으로 어려움을 겪고 있음을 확인했습니다. 이를 해결하기 위해 수강생들이 쉽게 고객 DB를 수집할 수 있도록 **랜딩페이지 자동 제작 도구**를 제공하고자 합니다.

### 1.2 프로젝트 목표
1. 수강생들이 코딩 없이 전문적인 랜딩페이지를 제작할 수 있게 함
2. 마그네틱 세일즈 철학이 녹아든 AI 기반 기획 도우미 제공
3. 고객 DB 수집을 위한 상세페이지 자동 생성
4. Claude 모델 기반의 고품질 AI 대화 경험 제공

### 1.3 타겟 사용자
- 마그네틱 세일즈 마스터클래스 수강생 (승인된 사용자)
- 보험 설계사, 피부샵 운영자, 코치/컨설턴트 등 1인 세일즈 종사자

## 5. 사용자 플로우

### 5.1 신규 사용자 플로우
```
회원가입 → 승인 대기 → 관리자 승인 → 로그인 → 대시보드
```

### 5.2 랜딩페이지 제작 플로우
```
1. 대시보드 → "랜딩페이지 만들기" 클릭
2. AI 기획 도우미와 대화 시작
3. AI가 질문 (업종, 타겟, 제공 가치 등)
4. 대화 완료 시 "프롬프트 생성" 버튼
5. 생성된 프롬프트 + 필요 이미지 목록 확인
6. 랜딩페이지 빌더로 이동
7. 프롬프트 붙여넣기 + 이미지 업로드
8. "생성" 버튼 클릭
9. 미리보기 확인 → 수정 → 배포
```

### 5.3 DB 수집 상세페이지 제작 플로우
```
1. 랜딩페이지 제작 완료 후 "상세페이지 추가"
2. AI가 상세페이지 기획 질문
3. 프롬프트 생성 → 빌더에서 생성
4. 구글 폼 URL 입력
5. 연결 버튼 자동 삽입
6. 배포
```
