```

   - 역할 정의
   - 전문성 범위
   - 경력/신뢰성 기술

3. Framework (프레임워크)
   - Python 클래스/함수 형태로 로직 정의

   - 다른 에이전트와의 연계
   - 핸드오프 조건

5. Configuration (설정)
   - 성능 지표
   - 품질 기준
   - 출력 형식

6. Deliverables (결과물)
   - 구체적 산출물 명세
   - 품질 체크리스트
```
